User Settings

As part of this phase, the following features will be provided in the module:

oProgram master: Program codes created here will reflect in the drop-down for Program code in the header section of an experiment. Landing page of this section will list existing program codes which may be edited by clicking the corresponding  (Pencil icon). A new program code may be created by clicking 'Add' button, typing the same name clicking 'add' here.

oTarget ID master: Target IDs created here will reflect in the drop-down for Target ID in the header section of an experiment. Landing page of this section will list existing target IDs which may be edited by clicking the corresponding  (Pencil icon). A new target ID may be created by clicking 'Add' button, typing the same name clicking 'add' here.
