Settings

Following features are provided as part of the user settings in this module:

oCustom Header Field: This feature has been provided in case the module needs an additional entry made which isn't available in the module's design. This will be treated as another column for all entries being made for any/all Equipment once the said custom header entry is added in the database. User may enter alphabets, number or special characters as values for the defined custom header.

Adding a new custom header entry

oEquipment Name: Equipment names may be added using this setting. An Equipment name will start reflecting in the Equipment master>Add Equipment>Equipment name drop-down once it is added here. User may enter alphabets, number or special characters as values here.

Adding a new Equipment type

oLocation: Placement locations may be added to the list here. These will start reflecting in the Equipment Master>Add Equipment>Location drop-down once it is added here. User may enter alphabets, number or special characters as values here.

Adding a new location to the list
