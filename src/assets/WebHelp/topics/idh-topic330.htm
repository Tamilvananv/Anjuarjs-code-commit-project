Common features and Notes

 

 

 

Common features on all pages of the application

The below list of features is present on a fixed panel (top horizontal section, left vertical menu bar). These features are accessible during use of the module.

❖Top horizontal section:

ØLogo of the company, on click, takes the user to the application module master.

ØThree horizontal lines next to the company logo, on click, expand/collapse the left vertical menu bar. These appear on selection of one of the modules listed on the application module master.

ØCenter of this section displays page name.

ØHelp section may be accessed by clicking on the Question mark icon. This will list all the topics on how to operate the module.

ØNotification feature (Bell Icon) pops up alerts, updates and notifications.

ØUser settings feature, on click, pops up has the below options:

▪Link to 'logout' of application

ØBreadcrumb trails (e.g. Home > User Management) are placed below the three horizontal lines on top left to keep track and maintain awareness of location on the web page.

❖Left vertical Menu Bar:

ØThis Menu bar, on moving the pointer on the bar, expands to show the menu bar with options to navigate on click.

❖Additional features common on all pages:

ØMandatory fields on all pages will be highlighted with an Asterisk and information (i) will be provided for labels/headers for more clarity wherever deemed necessary (in upcoming patch deployments)

ØAudit trail is captured for all modules of the project in this module. This is applicable to this module internally as well.

ØSearch and filter options are provided for each module individually.

ØButtons commonly used throughout the module:

▪Save: This button, on click, will save entry in the database. This will also make an entry in the audit trail of the module.

▪Next: This button, on click, will route the user to the next screen in flow of application.

▪Back: This button, on click, will route the user to the previous page in flow of the application.

▪Cancel: This button, on click, will cancel the action being performed and route the user to landing page of the current section.

▪Add: This button, on click, will add a record in the corresponding folder and in the database.

▪Browse: This button, on click, will route the user to the file manager of the local directory.

▪Export: This button, on click, will Export results from the given page on the local machine in selected format for reporting purposes.
