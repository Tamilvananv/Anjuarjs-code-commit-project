Introduction

Welcome to the Administration module. This module is designed to assist in managing Users, User Privileges, Departments and Audit Trails in a systematic and traceable manner. Data populated in this section is universal to all modules within the Digification project and entries once added here will start reflecting in those modules too. This module may be used to track user activity and history throughout the lifespan of this system. Audit trails section of the module may also be accessed through the Audit management section of the Electronic Quality Management System (E-QMS) module for referral. This module is linked with Immuneel's Active directory to export primary user data for logging in to the portal.

This guide will provide detailed information of the requested behavior and how the system will function. It is created based on the high-level requirements identified in the Business Requirements (URS) Document and provides traceability of the functional specifications back to the the user requirements.
